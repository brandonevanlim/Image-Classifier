# Image Classifier: Convolutional Neural Network

A CNN trained to predict image from real handwritten images by humans.

## Setup

```bash
pip install numpy pillow scikit-learn matplotlib tensorflow

**or**

pip install numpy
pip install pillow
pip install scikit-learn
pip install matplotlib
pip install tensorflow
```

## Get Training Data

Download a npy file from Kaggle:
https://www.kaggle.com/datasets

Place the file at the root. 

## Train

```bash
Just run the file as it has already been written
```
